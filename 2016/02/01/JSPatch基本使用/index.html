JSPatch基本使用

  • iOS
iOS JSPatch
本文总阅读量次

由于iOS项目上架AppStore的审核周期较长，而 JSPatch 能实现使用JS代码，替换项目原生代码修复bug，实现对项目的热更新，意义由此可见。
具体JSPatch原理可自行搜索查看

  1. pod引入JSPatch

    pod 'JSPatch'

  2. AppDelegate中引入JSPatch头文件

    #import <JPEngine.h>

  3. 一般在App进入时，根据版本号等，执行网络请求，获取需要执行的js代码

  4. 执行js代码

    [JPEngine startEngine];    //开启引擎
[JPEngine evaluateScript:js];//执行js代码

oc转js代码

oc 无参数

- (void)viewDidLoad {
    [super viewDidLoad];
    self.title = @"发现";
    _selectedIndex = 0;
    [self initView];
}

js

defineClass('FindViewController',{
    viewDidLoad: function(){
        self.super().viewDidLoad();
        self.setTitle('发现哈哈');
        _selectedIndex = 0;
        self.initView();
    }
})

oc 有参数

require('UIAlertView');defineClass('CircleHotPageViewController',{
    tableView_didSelectRowAtIndexPath:function(tableView,indexPath){
        tableView.deselectRowAtIndexPath_animated(indexPath, YES);
            var alert = UIAlertView.alloc().initWithTitle_message_delegate_cancelButtonTitle_otherButtonTitles('只是提示而已', '测试JSPath！', null, '知道了', null, null);
            alert.show();
        }
})

js

- (void)tableView:(UITableView *)tableView didSelectRowAtIndexPath:(NSIndexPath *)indexPath {
[tableView deselectRowAtIndexPath:indexPath animated:YES];

    CircleHotModel *model = _hotDataArray[indexPath.row];
    CircleDetailViewController *detailVC = [[CircleDetailViewController alloc] init];
    detailVC.srcTitle = model.title;
    detailVC.cid = model.cid;
    [self pushVC:detailVC animated:YES];
}

oc 在一个方法后，添加代码

- (void)refreshAction {
    NSLog(@"刷新。。");
    [self getData];
}

js

require('UIAlertView');defineClass('HomePageVCNew',{
    refreshAction: function() {
        self.ORIGrefreshAction();
        var alert = UIAlertView.alloc().initWithTitle_message_delegate_cancelButtonTitle_otherButtonTitles('测试js', '哈哈', null, '取消', null, null);
        alert.show();
    }})

self.ORIGrefreshAction() ，表示执行之前已有的方法
